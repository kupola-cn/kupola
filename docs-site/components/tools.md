# 工具组件

这一组组件是全局工具、数据展示和表格能力的集合。

```js
import { Icons } from '@kupola/kupola/components/icons'
```

## 组件清单

| 组件 | 导入 | 说明 |
| --- | --- | --- |
| Icons | `@kupola/kupola/components/icons` | SVG 图标集 |
| Message | `@kupola/kupola/components/message` | 轻量级全局消息 |
| Heatmap | `@kupola/kupola/components/heatmap` | 热力图 |
| Table | `@kupola/kupola/components/table` | 数据表格 |

## 重点

- `Message` 适合轻量成功/失败反馈。
- `Table` 是库里最常用的数据容器之一，支持排序、分页、选择和编辑。
